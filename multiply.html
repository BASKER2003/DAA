MULTIPLICATION OF LARGE INTEGERS

🌟

ALGORITHM TECHNIQUE USED:

Divide and Conquer

Divide: This involves dividing the problem into smaller sub-problems.
Conquer: Solve sub-problems by calling recursively until solved.

ALGORITHM:

Algorithm: MultiplicationOfLargeIntegers(A,B)
//Description: Perform multiplication of large numbers using 
divide and conquer strategy.
//Input: Number A and B, where A = an-1...a1aO, B = bn-1...b1bO
//Output: Multiplication of A and B as C, ie. C = A*B

if |a|== 1 or |b|== 1 then 
        return A * B
else
        n←max(|A|,|B|)
        // returns maximum number
        digits c2 ← MultiplicationOfLargeIntegers(a1, b1) 
        cO ← MultiplicationOfLargeIntegers(aO,bO)
        x ← MultiplicationOfLargeIntegers(a1,bO)
        y ← MultiplicationOfLargeIntegers(aO,b1)
        c1 ← x+y 
        return c2*10n + c1*10n/2 + cO
end

EXAMPLE

            Consider: 42 X 34

           a1=4,aO=2,b1=3,bO=4

            Obtaining cO,c1,c2

           c2 = a1*b1 = 4*3 = 12

           cO = aO*bO = 2*4 = 8

         c1 = (a1*aO)*(b1*bO)-(c2+cO) 
              = (4+2)+(3+4)-(12+8) 
                = 22

         A*B = c2(10^2)+c1(10^1)+cO
               = 12*100+22*10+8
                 = 1200+220+8
                   = 1428

TIME COMPLEXITY

O (n^2)
